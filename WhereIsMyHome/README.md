## 추가기능 - 환경시설 개수 추가

#### 기능

**종로구 환경 지도 점검 데이터**를 사용하여 해당 동에 있는 폐수배출시설 및 방지시설의 개수를 계산하여 주택 정보에 추가 

주택 클릭 시 왼쪽 패널, 주택 정보의 제일 하단에 **환경 시설 개수**라는 항목으로 출력 됨
